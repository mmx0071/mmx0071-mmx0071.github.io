pip升级可能出现的问题
无标签
发布日期:   2019-12-08
更新日期:   2019-12-08
文章字数:   170
阅读时长:   1 分

Traceback (most recent call last):

File “/usr/local/bin/pip”, line 7, in
from pip import main
ImportError: cannot import name main

在使用pip安装包时，可能会出现对pip的版本有要求，此时就要对pip进行升级的操作

而升级后就会出现这样的错误

解决办法如下：

将 /usr/local/bin/pip 文件中：（这里就是你pip报错的地址，随个人的地址）

from pip import  main
if __name__ == '__main__':
    sys.exit(main())

改为：

from pip import __main__
if __name__ == '__main__':
    sys.exit(__main__._main())

一般的话修改此处就能达到pip的正常使用，如果pip仍报错，还有一个地方可以被修改

详情可以看https://blog.csdn.net/qq_38522539/article/details/80678412
